Slack Clone App
============================
This repository hosts an MVP of the Slack app.

## Project structure
1. system login (only sign in and use fake data user)
2. sidebar on the left:
  + list of channels
    + add / remove button
  + list of users - direct messages
3. chat window on the right
  + list of messages
    + each message
      + avatar
      + username
      + timestamp
      + message
      + edit / remove button
  + comment box

## Tech stack
1. react (+ jsx)
2. redux (+ storage)
3. react-router
4. Moment
5. Daisy UI, Tailwind.
